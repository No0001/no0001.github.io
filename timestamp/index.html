What is a Unix Timestamp?

      
A Unix timestamp (also known as Epoch time or POSIX time) is the number of seconds that have elapsed since January 1, 1970, 00:00:00 UTC (the Unix Epoch). It is a widely used standard for tracking time in computing systems.


      
Seconds vs Milliseconds

      
Unix timestamps are traditionally measured in seconds (10 digits, e.g., 1700000000). However, many programming languages and APIs use milliseconds (13 digits, e.g., 1700000000000). This tool auto-detects the format based on the number of digits.
