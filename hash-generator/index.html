What is a Hash Function?

      
A cryptographic hash function takes an input (or "message") and produces a fixed-size string of bytes, typically displayed as a hexadecimal number. The output is called a hash, digest, or checksum. Hash functions are one-way â€” you cannot reverse a hash back to the original input.
